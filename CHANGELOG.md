# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2025-01-04

### Added

- Initial release
- Cross-platform achievement abstraction layer
- Built-in providers: Local, Steam, Game Center, Google Play
- Visual editor dock for managing achievements
- Toast notification system with customizable scenes
- Progress tracking for incremental achievements
- Import/Export achievements (JSON, CSV)
- Code generation for type-safe achievement constants
- Custom properties system for extensibility
- Local-first persistence with platform sync
